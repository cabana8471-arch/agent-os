# Agent OS - Reguli Generale

Framework spec-driven pentru agenți AI. Bash scripts + Markdown templates.

## 6 Faze de Dezvoltare
plan-product → shape-spec → write-spec → create-tasks → implement-tasks → orchestrate-tasks

## Comenzi Principale
- `~/agent-os/scripts/project-install.sh` - Instalare în proiect
- `~/agent-os/scripts/project-update.sh` - Actualizare instalare
- `~/agent-os/scripts/create-profile.sh` - Creare profil nou

## Profile Disponibile
default, nextjs, react, seo-nextjs-drizzle, wordpress, woocommerce

## Structură Proiect
- `profiles/` - Profiluri configurabile cu agents/, commands/, standards/, workflows/
- `scripts/` - Scripturi Bash principale
- `config.yml` - Configurație globală
